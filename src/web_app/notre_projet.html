Projet IOT & ML

Dans le cadre d'un cours de Machine Learning & IoT, nous avons réalisé un projet de surveillance d'examen.
Ce projet a été réalisé par Traian BATOG (traianbatog@yahoo.com) et Aghilas KESSAÏ (aghilaskessai@yahoo.com) à l'Université Sorbonne Paris Nord encadré par Aomar OSMANI et Massinissa HAMIDI. Pour en savoir plus, vous pouvez consulter la page Github du cours d'IOT avec tous les projets réalisés par les étudiants.

Les outils et les technologies utilisées :

  • Web : Javascript, HTML, CSS, Bootsrap
  • Communications entre les téléphones et le PC: gRPC
  • Application Android : Java & Kotlin
